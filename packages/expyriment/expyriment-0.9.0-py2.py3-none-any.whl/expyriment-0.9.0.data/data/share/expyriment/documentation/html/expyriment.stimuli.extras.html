expyriment.stimuli.extras¶

The stimuli extra package.

Notes¶

Extra stimuli classes are available via expyriment.stimuli.extras.*. (For expample: expyriment.stimuli.extras.DotCloud)

To us the extras module you have to import it manually by calling: import expyriment.stimuli.extras

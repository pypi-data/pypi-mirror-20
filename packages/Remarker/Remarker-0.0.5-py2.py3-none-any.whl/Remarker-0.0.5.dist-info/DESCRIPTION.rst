A command line tool for generating Remark.js presentations from markdown files.

* Free software: MIT license
* Documentation: (COMING SOON!) https://remarker.readthedocs.org.

Usage
-----

See ``remarker --help``

Usage Example
-------------

``remarker -o presentation.html slides.md``
